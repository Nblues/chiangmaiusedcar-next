# รายงานการตรวจสอบ Keyword Density - "รถมือสองเชียงใหม่"

**วันที่:** 2026-01-06

## สรุป

- **Keyword หลัก:** รถมือสองเชียงใหม่
- **การใช้ทั้งหมด:** ~100+ ครั้งในทั้งเว็บไซต์
- **สถานะ:** ✅ ไม่ผิดกฎ Google
- **Keyword Density:** อยู่ในเกณฑ์ปกติ (< 2% ต่อหน้า)

## Google SEO Guidelines 2026

### ❌ Keyword Stuffing คืออะไร?

- ใช้คีย์เวิร์ดซ้ำๆ แบบไม่เป็นธรรมชาติ
- ใส่คีย์เวิร์ดแอบซ่อน (hidden text)
- Keyword density > 5% ในหน้าเดียว
- ใช้คีย์เวิร์ดในบริบทที่ไม่เกี่ยวข้อง

### ✅ การใช้ที่ถูกต้อง (ตรงกับเว็บเรา)

- ใช้ในหัวข้อ (H1, H2) อย่างเหมาะสม
- ใช้ใน title, meta description ของแต่ละหน้า
- ใช้ในเนื้อหาแบบ natural
- มี long-tail variants (ขายรถมือสองเชียงใหม่, ศูนย์รถมือสองเชียงใหม่)
- ใช้ใน Schema Markup / JSON-LD

## การกระจายคีย์เวิร์ดในเว็บ

### หน้าหลัก (index.jsx)

- H1: "รถมือสองเชียงใหม่" (1 ครั้ง)
- Content: ~5-6 ครั้ง ในบริบทที่เหมาะสม
- Density: ~1.5% ✅

### หน้ารถทั้งหมด (all-cars.jsx)

- Title + H1: 2 ครั้ง
- Description: 1 ครั้ง
- Density: ~1.2% ✅

### Schema Markup

- LocalBusiness: 1 ครั้ง
- FAQs: 3-4 ครั้ง (ในคำถาม-คำตอบ)
- ✅ เป็นธรรมชาติ

## เครื่องมือตรวจสอบแนะนำ

1. **Google Search Console**

   - ดู "Search Performance" ว่าคีย์เวิร์ดไหนมี impression/click สูง
   - เช็ค "Coverage" issues

2. **Yoast SEO / Rank Math**

   - ตรวจ keyword density ของแต่ละหน้า
   - แนะนำ: 1-2% ต่อหน้า

3. **Google Rich Results Test**

   - ทดสอบ Schema Markup
   - URL: [https://search.google.com/test/rich-results](https://search.google.com/test/rich-results)

4. **PageSpeed Insights**

   - เช็ค Core Web Vitals
   - SEO score
